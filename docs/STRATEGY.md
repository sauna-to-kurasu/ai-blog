### なぜやるべきか（リサーチ結論）
- **市場の空白**: 「Claude Code 使い方」ハウツーは企業メディアが飽和だが、「非エンジニアの実運用ドキュメンタリー」はほぼ不在
- **追い風**: CloudflareがAstroを2026年1月に買収。注目度急上昇中
- **超低コスト**: .devドメイン約1,550円/年 + Cloudflare Pages無料 = 年間約1,550円
- **資産の再利用**: サウナブログの80-90%を流用可能。2-3週間で立ち上げ可能

### 決定事項
- ブログ名: **「AIと暮らす。」**（ai-to-kurasu.dev）
- Tailscale+Termius記事: **Phase 1に前倒し**（キラーコンテンツ）

### 差別化の3軸
1. **リアルタイム運用記録**: サウナブログをClaude Codeで構築・運用している過程そのものがコンテンツ
2. **定量データの徹底**: 「Lighthouse 100点」「運用コスト年間1,550円」など数字で語る
3. **モバイル開発ワークフロー**: Tailscale + Termius + Claude Codeの体系的な情報発信（空白地帯）

### コモディティ化リスクへの認識（2026-03-31 エージェントチーム議論）

**ハウツー記事は6-12ヶ月で飽和する。** 「Claude Codeの使い方」系は企業メディアが量産中であり、AIツール自体の進化で「教え方」コンテンツの価値は下がる。

**しかし以下は残る：**
- **完走率の壁**: 「やり方を知っている」と「ゼロからやり切って運用している」の間には大きなギャップがある。この壁が本物の差別化
- **時間と実績の参入障壁**: 85エントリの開発ログ、2サイトの実運用実績、コミット履歴は後から作れない。物理的参入障壁（サウナブログ）より薄いが、先行者利益として機能する
- **ナラティブの不可複製性**: 「30代共働き子育て非エンジニアがやり切った」というストーリーは唯一無二。ただしナラティブは集客装置であり、収益化エンジンは商品力に依存する

**戦略方針**: 大きく賭けるのではなく、低コストで実験し小さく回収する設計。撤退コスト年間1,550円を活かす

### 市場分析の根拠
- 梶谷健人氏のnote「Claude Codeはエンジニア以外も全員使うべき」が大きくバズ
- 日経クロストレンドが「非エンジニアこそ生産性に差がつく」と特集
- Zennで「週刊AI駆動開発」連載が定着
- 「Claude Code 使い方」は企業メディア（DevelopersIO、AI Native、GIG等）が飽和
- **空白地帯**: 非エンジニアが実際にClaude Codeで運用しているブログの裏側を公開する体験記

## 4. 収益化ロードマップ（自前プラットフォーム戦略）

### 競合の価格帯（参考）
- claude-code-academy.dev: **24,900円**（買い切り、Next.js + Stripe + Bunny CDN）
- Claude Code マスター講座: **29,800〜98,000円**（3プラン）
- Udemy Claude Code講座: 定価12,800円（セール時1,800円）

### Udemyを避ける理由
- 2026年時点でUdemyのサブスクシフトにより**講師取り分がわずか15%**に悪化
- 全社売上の74%がサブスク経由。単発コース販売を縮小する方針
- 海外トップ講師が50-70%の収益減を報告し、自前プラットフォームへ移行中
- 顧客リストを持てない（メールアドレスはUdemyのもの）

### 自前プラットフォームの構成

```
[ユーザー] → Cloudflare Pages/Workers (Astro SSR)
              ├── Stripe Checkout → 決済（手数料3.6%のみ）
              ├── Cloudflare D1 → ユーザー/購入管理
              ├── Cloudflare KV → セッション
              └── Cloudflare Stream → 動画配信（署名付きURL）
```

月額コスト: **約$5-10**（Cloudflare Stream + Workers）

### 「プラットフォーム自作 = コンテンツ」の入れ子構造

**Claude Codeで講座プラットフォームを作り、その過程をClaude Code講座として売る。** 受講者はまさにその成果物の上で学んでいる。

> **2026-03-31 修正**: 入れ子構造は「講座の核心」ではなく「マーケティング上の信頼性証拠」として活用する。受講者が求めるのは「自分のブログを作れること」であり、プラットフォームの作り方ではない。講座記事（No.14, 15）はブログコンテンツとしては有効だが、講座カリキュラムとは切り離す。

- 非エンジニアが作った事実が最大の証明（信頼性の担保）
- 構築過程はブログ記事 + YouTube動画として発信
- **プラットフォーム構築は後回し**: まずGumroad/STORESでMVP販売し需要を検証。自前プラットフォームはその後

### 商品ラインナップ（3段階）

> **2026-03-31 修正**: note(¥980)→講座(¥15,000)の価格断崖を埋めるため、スターターキットを中間商品として新設。講座価格も¥14,800に調整。

### 4.1 講座カリキュラム

> サウナブログの `COURSE_OUTLINE.md` から統合（2026-03-30）。
> 自前プラットフォーム（Cloudflare Stream）での動画配信を前提に再構成。

**タイトル案:**
「WordPress はもう古い？コード経験ゼロでも AI に話すだけで爆速ブログが作れる講座」

**副題:**
Claude Code × Astro × Cloudflare で、シミュレーターも作れる本格サイトを構築

**講座の価値:**
WordPress をポチポチ設定する時代は終わった。AI に口頭で伝えるだけで、WordPress では難しいシミュレーターや爆速サイトが作れる。

**ターゲット:**
忙しい30代・子育て世代で副業ブログを始めたい人。WordPress は面倒、でも自分のサイトは持ちたい。IT知識は問わない。

**講師の説得力:**
プログラミング経験ゼロ。会社でもコードに触れたことがない。基本情報処理技術者の資格は持っているが、正直それは関係なかった。大事なのは「やりたいことを言葉にできるか」だけ。

#### WordPress との差別化（講座内で必ず見せる）

| | WordPress | この講座の方法 |
|---|---|---|
| 設定 | 管理画面をポチポチ | 口頭で伝えるだけ |
| シミュレーター等 | プラグイン探し or 外注 | 「こういうの作って」で完成 |
| 表示速度 | プラグインで重くなりがち | 爆速（静的生成・Lighthouse 100点） |
| 運用コスト | サーバー月1,000円〜 | Cloudflare 無料枠で0円 |
| 忙しい人向き | 自分で設定する時間が必要 | AI に丸投げできる |

### サウナブログの戦略的活用

サウナブログは「実績の証拠」にとどまらず、AIブログの収益化を**能動的に支援する資産**として活用する。

1. **講座の信頼材料**: 講座LPの実績セクションにサウナブログのスクショ・PVデータ・Chart.jsグラフ等を掲載。「非エンジニアがここまで作れる」の証拠
2. **体験記の素材**: 「体験記」カテゴリでサウナブログの構築プロセスを取り上げる（シミュレーター開発、Chart.js実装、CLAUDE.md育成等）
3. **スターターキットの実例**: スターターキットに含むテンプレート・ルールの「実際に使われている」サイトとしてサウナブログを提示

### 相互リンク
