> 作成日: 2026-03-29
> Claude Codeエージェントチームによる市場リサーチ・競合分析・戦略設計の結果

## Context

サウナブログ「サウナと暮らす。」を運営する非エンジニアのユーザーが、Claude Code + Astro + Cloudflare Pages + Tailscale/Termiusでブログを作るノウハウを発信するテック系ブログを立ち上げる。今週末から着手。

### なぜやるべきか（リサーチ結論）
- **市場の空白**: 「Claude Code 使い方」ハウツーは企業メディアが飽和だが、「非エンジニアの実運用ドキュメンタリー」はほぼ不在
- **追い風**: CloudflareがAstroを2026年1月に買収。注目度急上昇中
- **超低コスト**: .devドメイン約1,550円/年 + Cloudflare Pages無料 = 年間約1,550円
- **資産の再利用**: サウナブログの80-90%を流用可能。2-3週間で立ち上げ可能

### 決定事項
- ブログ名: **「AIと暮らす。」**（ai-to-kurasu.dev）
- Tailscale+Termius記事: **Phase 1に前倒し**（キラーコンテンツ）
- 着手時期: **今週末から即開始**
- 収益化: **自前プラットフォーム志向**（Udemyではなく自作の講座サイト）

## 4. 収益化ロードマップ（自前プラットフォーム戦略）

### 競合の価格帯（参考）
- claude-code-academy.dev: **24,900円**（買い切り、Next.js + Stripe + Bunny CDN）
- Claude Code マスター講座: **29,800〜98,000円**（3プラン）
- Udemy Claude Code講座: 定価12,800円（セール時1,800円）

### Udemyを避ける理由
- 2026年時点でUdemyのサブスクシフトにより**講師取り分がわずか15%**に悪化
- 全社売上の74%がサブスク経由。単発コース販売を縮小する方針
- 海外トップ講師が50-70%の収益減を報告し、自前プラットフォームへ移行中
- 顧客リストを持てない（メールアドレスはUdemyのもの）

### 自前プラットフォームの構成

```
[ユーザー] → Cloudflare Pages/Workers (Astro SSR)
              ├── Stripe Checkout → 決済（手数料3.6%のみ）
              ├── Cloudflare D1 → ユーザー/購入管理
              ├── Cloudflare KV → セッション
              └── Cloudflare Stream → 動画配信（署名付きURL）
```

月額コスト: **約$5-10**（Cloudflare Stream + Workers）

### 「プラットフォーム自作 = コンテンツ」の入れ子構造

**Claude Codeで講座プラットフォームを作り、その過程をClaude Code講座として売る。** 受講者はまさにその成果物の上で学んでいる。これは極めて説得力がある。

- 非エンジニアが作った事実が最大の証明
- エラー、Cloudflareの設定ハマり、Stripeの設定、認証の実装 -- すべてが講座コンテンツになる
- メイキング動画はYouTubeで集客コンテンツになる

### 収益ファネル

```
[認知] テックブログ「AIと暮らす。」（SEO流入、無料）
  ↓ 記事末尾CTA
[検討] 有料note / Zenn記事（500-980円）
  ↓
[購入] 自前プラットフォームの講座（15,000-25,000円、買い切り）
  ↓
[信頼] サウナブログ（実績の証拠、「この人は本当にやっている」）
```

### フェーズ別計画

**Phase 1（0-3ヶ月）: ブログ基盤**
- ブログ立ち上げ + 初期10記事
- AdSense申請、Amazon/楽天アフィリエイト登録
- 目標収益: 0円（投資期間）

**Phase 2（3-6ヶ月）: 初期収益化 + プラットフォーム構築開始**
- 有料note/Zenn記事 1-2本（500-980円）
- 自前講座プラットフォーム構築開始（Claude Codeで）
- **構築過程をブログ記事 + YouTube動画として発信**
- 目標収益: 月3,000-5,000円（AdSense + note）

**Phase 3（6-12ヶ月）: 講座販売開始**
- 自前プラットフォームで講座販売（15,000-25,000円買い切り）
- YouTube メイキング動画 → 講座への導線
- 目標収益: 月15,000-50,000円
  - AdSense: 月3,000-5,000円
  - 有料note/Zenn: 月3,000-5,000円
  - 自前講座: 月10,000-40,000円（月1-2本販売）

## 5. サウナブログとの関係

### 「同じ人が運営する別ブランド」戦略
- ドメインは完全に別（sauna-to-kurasu.com / ai-to-kurasu.dev）

- 筆者プロフィールは共通

- **テック→サウナ**: 「サウナブログのこの機能をClaude Codeで作った裏側」
- **サウナ→テック**: プロフィールに導線、「ツール＆アプリ5選」記事から誘導
