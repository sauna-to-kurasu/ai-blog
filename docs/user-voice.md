### 2026-04-02 / WordPressからAI+Astroへの体験変化

- **やったこと**: 2026年1月にWordPressでブログを始めたが、途中からClaude Code + Astro + Cloudflareに移行
- **感想**: 「昔、WordPressでブログを作っていたときは、本当にすごく時間がかかっていた。記事の構成を考えたり、考え終わったらそれをマウスでぽちぽちWordPressでデザインを整えたりとか。ただ、それも結局は自分がいいなと思っているだけで、本当のプロの目線ではどうかというところが欠けていた。今は自分がいいなというのは選びつつも、プロ目線での提案の中から選べていく。そういったところがすごく魅力。これはもうWordPressでは無理」
- **温度感**: 非常に高い。WordPress時代との比較から来る実感
- **核心的洞察**: AIの価値は「自動化」だけでなく「プロ品質の選択肢を提示してくれること」。非エンジニアにとって、自分のセンスだけに頼らずプロレベルの提案から選べる体験が最大の差別化
- **関連記事候補**: 講座の訴求全体、No.1体験記の深掘り版、Phase 2「AIに聞けば済む時代に、なぜ僕はブログを書くのか」

### 2026-04-02 / 「AIに聞けば済む時代にハウツー記事は必要か？」

- **温度感**: 中〜高。懐疑的だが建設的。戦略を改善したいという意思

### 2026-04-03 / Cursor → Claude Code 乗り換え体験

- **やったこと**: 最初はClaude Pro（$20）+ Cursor Pro（$20）= 月$40で運用。Claude chatでプロンプトを作成し、それをCursorに投げるワークフロー。Cursorではプロジェクトフォルダを丸ごと読み込める点が魅力だった。その後Claude Codeに移行し、Cursor課金を解除
- **感想**: 「Cursorの中でもClaudeが一番コード生成能力が高かった。でもすぐ上限に達して、1ヶ月リセット待ち。知らないうちに課金されていたのもつらかった。結局Claude Codeで同じことができるとわかって、じゃあもうCursor課金しなくていいねとなった」
- **温度感**: 高い。実体験に基づく明確な比較判断
- **関連記事候補**: No.7「CursorからClaude Codeに乗り換えた非エンジニアの話」

### 2026-04-03 / Googleアプリ経由の契約は高い

- **やったこと**: 最初はGoogleアプリ（Gemini）経由でClaude Maxプラン（$100）を契約。その後Anthropic直接契約に変更
- **感想**: 「Googleアプリ経由だとすごい高かった。$100プランが通常15,000円くらいのところ、2万円くらいした。Googleの取り分が乗っかってくるらしい。直接契約のほうが安いので伝えておきたい」
- **温度感**: 中〜高。読者に伝えたいという意思あり
- **関連記事候補**: No.7、または独立したTips記事
