- `/login`しないとAPI従量課金モードになる: 事実
- GitHub PAT認証: スコープは`repo`（Geminiの指示通り）。PATはPCで生成→Google Keepにメモ→スマホからコピペ

- Amphetamine設定: 「ログイン時に起動」「自動セッション開始」「スリープ解除時も自動セッション」すべてON。常時無制限セッション
- macOSバッテリー設定: 「電源アダプタ使用時はディスプレイがオフのときに自動でスリープさせない」ON
- `sudo pmset -c disablesleep 1`: Geminiメモでは「検討」段階。実行した可能性あり
- **最終的なスリープ対策**: MacBookを完全に閉じず1-2cm開けておく運用。完全に閉じると対策①〜③を講じても通信が切れる
- Amphetamine常時起動のハードウェアへの影響: AIに確認済み、ほぼ問題なしとの回答
- Termiusで日本語入力・音声入力ができない: 事実。唯一にして最大の欠点（今でもそう感じている）
- Google Keepメモで音声入力→コピー→Termius貼り付け: 実際のワークフロー

- 日本語入力さえ解決すれば、左Termius / 右Chrome でリアルタイムプレビュー確認もできるはず
- GA4/GSC: MCP接続済み。スマホ/PC両方から「サウナブログの分析をして」等の指示で自動分析
